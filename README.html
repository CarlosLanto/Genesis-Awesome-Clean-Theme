Installation Instructions

  1. Upload the theme folder via FTP to your wp-content/themes/ directory. (The Genesis parent theme needs to be in the wp-content/themes/ directory as well.)
  2. Go to your WordPress dashboard and select Appearance.
  3. Activate the Genesis Awesome Child Theme
  4. Inside your WordPress dashboard, go to Genesis > Theme Settings and configure them to your liking.

SCSS Mixins

  • Rem mixin. Adds a element whit rem values

    .class{
 @include rem (margin, 10px 20px);
}

out:
.class{
  margin: 10px 20px;
  margin: 1rem 2rem;
}
